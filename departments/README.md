# Departments

Within Unmatched RP there are different departments within Staff. This allows each department to deal with their own specific area within the server and to maintain it to the correct level and standard that we require.

In this section it will outline what all the departments are, what they do, who heads up the departments, the escalation process within those departments & also escalation beyond Management.

Please take the time to read over this carefully and remember what each department does.
