# Handling Reports

Handling Reports is one of the most important areas within the server & community. Without proper handling of reports the community have no trust or faith in the staff teams. We must treat every report the same way regardless of the situation so that we can confidently say at the end of it that everyting was handled correctly.



1. Read the report.
2. Gather evidence.
3. Speak to all parties.
4. Review logs.
5. Make an impartial decision.
6. Explain the outcome.
7. Document the action.
